# SSVEP

This repository provides notes, methodologies, and reference materials for working with Steady-State Visually Evoked Responses (SSVEP), including analysis techniques and experimental considerations.

# Used Datasets

1. [MAMEM Project Datasets](https://www.mamem.eu/results/datasets/)

2. [8-Channel SSVEP EEG Dataset with Artifacts Trials](https://openneuro.org/datasets/ds004745/versions/1.0.0)

3. [Multi-frequency steady-state visual evoked potential dataset](https://www.nature.com/articles/s41597-023-02841-5?fromPaywallRec=false#Sec17) ([access](https://figshare.unimelb.edu.au/articles/dataset/Multi-Frequency_Steady-State_Visual_Evoked_Potential_Dataset/22015694))
