# SSVEP

This repository provides notes, methodologies, and reference materials for working with Steady-State Visually Evoked Responses (SSVEP), including analysis techniques and experimental considerations.

# Used Datasets


1. [Multi-frequency steady-state visual evoked potential dataset](https://www.nature.com/articles/s41597-023-02841-5?fromPaywallRec=false#Sec17) ([access](https://figshare.unimelb.edu.au/articles/dataset/Multi-Frequency_Steady-State_Visual_Evoked_Potential_Dataset/22015694))

2. [An open dataset for human SSVEPs in the frequency range of 1-60 Hz](https://www.nature.com/articles/s41597-024-03023-7) ([access](https://springernature.figshare.com/collections/An_open_dataset_for_human_SSVEPs_in_the_frequency_range_of_1-60_Hz/6752910/1))

3. [MNE SSVEP Dataset](https://mne.tools/stable/documentation/datasets.html#ssvep-dataset)
